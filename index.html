Intuitive Website Management Tool

GDS

For members that opt for the annual subscription, we will provide you with your own Galileo or Amadeus GDS terminal. You will be able to use all the functionality of a GDS including checking fares and rules, searching for availability, holding seats, creating PNRs and issuing tickets.

For members that opt for the monthly subscription, you will have access to the GDS emulator which has all the native functionality of the GDS terminal using commands but without some of the UI assistance features that are offered on Galileo Smart Point or Amadeus Web Terminal.

The GDS is the system used by all airlines to store and distribute their fares and availability as well as create live bookings directly into the airline system.

The GDS is especially useful for those that now how airlines manage their yield management. Lower fare booking classes are often released at odd times of the day, but do follow a pattern. Often, this is in the middle of the night. So if you check availability at the right time, you can often save a lot of money on your flight tickets.

Redemption Flights: If you know which class the airline uses for redemption tickets, you can check availability much faster using the GDS

Print Vouchers

If you interact with your customers in person, the giving a beautifully designed printed discount voucher will help you drive your customers to book travel using your travel brand. We have made this really easy for you in a few simple steps:

  • Log on to your portal
  • Select a voucher design
  • Set the primary secondary and tertiary colours
  • Set the discount amount (percentage or value)
  • Set any rules such as £100 off any multi centre holiday booking over £3,000
  • Set expiry date
  • Select the quantity you require

Print Vouchers

The process is as simple as the print voucher. The only difference is, once you have created your voucher, you need to email the voucher to the client. The steps are:

  • Create the voucher by setting the amount and rules
  • Enter as many client email addresses as you want to send the voucher to
  • Include the online voucher in an email newsletter or mail shot campaign
